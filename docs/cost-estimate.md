## 4. OpenAI API費用（参考）

GCPの費用ではないが、運用コストとして参考値を記載する。

**使用モデル想定**: GPT-4o-mini（コスト重視）  
**料金**: Input $0.15 / 1M tokens・Output $0.60 / 1M tokens

| 前提 | 値 |
|---|---|
| MAU | 50人 |
| 1ユーザーあたりの会話ターン数 | 10ターン / 日 |
| 月間総ターン数 | 50人 × 10ターン × 30日 = 15,000ターン |
| 1ターンあたりのトークン数（Input） | 600 tokens（システムプロンプト＋履歴＋ユーザー入力） |
| 1ターンあたりのトークン数（Output） | 300 tokens |

| 項目 | トークン数 | 月額（USD） | 月額（JPY） |
|---|---|---|---|
| Input | 15,000 × 600 = 9,000,000 | $1.35 | ¥209 |
| Output | 15,000 × 300 = 4,500,000 | $2.70 | ¥419 |
| **小計** | | **$4.05** | **¥628** |

> GPT-4o（高品質版）を使用する場合は約10倍のコストになる（$40〜$50 / 月）。

## 6. スケールアップ時の費用推移

| フェーズ | MAU | GCP月額 | Apple月割 | OpenAI月額 | 合計月額 |
|---|---|---|---|---|---|
| MVP | 〜50人 | $9.50（¥1,473） | $8.25（¥1,279） | $4（¥628） | **$21（¥3,380）** |
| 成長期 | 〜500人 | $40（¥6,200） | $8.25（¥1,279） | $40（¥6,200） | **$88（¥13,640）** |
| 本格運用 | 〜5,000人 | $150（¥23,250） | $8.25（¥1,279） | $400（¥62,000） | **$558（¥86,490）** |

**成長期の主なGCPコスト増要因:**
- Cloud SQL: `db-f1-micro` → `db-g1-small` へ昇格（+$18/月）
- Cloud Run: min-instances=1 に変更（+$10/月）
- ストレージ拡張: 10GB → 50GB（+$7/月）

**本格運用時の主なGCPコスト増要因:**
- Cloud SQL HA（マルチゾーン）: 2倍のコスト
- Cloud CDN / カスタムドメイン: +$10〜$20/月
- Cloud Monitoring / Logging 強化
